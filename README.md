# Bėgimo Fiziologija — Interaktyvus Edukacinis Kursas

Vieno failo interaktyvus kursas pradedantiesiems bėgikams lietuvių kalba. 

## Turinys

8 nuoseklios pamokos:

1. **Kodėl bėgti lėtai?** — raumenų skaidulų tipai, 80% taisyklė
2. **Širdis ir treniruočių zonos** — MaksPP, Z1–Z5, pilkosios zonos spąstas
3. **Energija ir kuras** — riebalai, glikogenas, maratono siena
4. **Laktatas** — kodėl laktatas nėra priešas, laktatinis slenkstis
5. **Kvėpavimas** — nosis prieš burną, diafragminis kvėpavimas, VT1/VT2
6. **Karštis ir hidratacija** — širdies ritmo slinkis, gėrimo tvarkaraštis
7. **Bėgimo technika** — kadencija, perkrova per žingsnį, traumų prevencija
8. **Kaip tobulėti?** — superkompensacija, miegas, 10% gairė

Kiekvienoje pamokoje yra interaktyvus demonstravimas su slankikliais, kuris parodo fiziologinį principą veikiant — laktatinė kreivė, glikogeno bakas, karščio poveikio skaičiuoklė ir kt.
